# أكاديمية عايد الرسمية — برنامج تحديد مستوى STEP (V5 FINAL)

## التشغيل المحلي

1) افتح مجلد المشروع.
2) شغّل سيرفر بسيط:

```bash
python -m http.server 8000
```

3) افتح:

- `http://localhost:8000/index.html`

> ملاحظة: لازم سيرفر محلي (وليس فتح الملفات مباشرة) عشان الـ Service Worker والـ ES Modules.

## أهم الصفحات

- `index.html` — الرئيسية (تعريف + آيات/أحاديث + أزرار التثبيت/المشاركة + تقييمات + قصص)
- `start.html` — ملف المستخدم (الاسم + بياناتك) ثم انتقال للاختبار
- `test.html` — الاختبار الكامل (50 سؤال) + قفل 24 ساعة
- `results.html` — النتائج + خطة + جدول + نسخ + مشاركة + تحميل PDF
- `quiz.html` — الكويزات (توليد + Seed + تشغيل + مراجعة أخطاء + مشاركة)
- `progress.html` — السجل (محاولات + كويزات + حذف بيانات المستخدم)
- `faq.html` — FAQ (يقرأ من JSON)
- `group.html` — مجموعة الالتزام (واتساب/تلجرام) + رسالة مشاركة جاهزة
- `support.html` — نموذج دعم (يرسل إلى `stepacademy438@gmail.com`)

## بيانات المحتوى (JSON)

كل ملفات المحتوى الجاهزة في:

`assets/content/`

- `notifications.json` — 900+ إشعار (Toast)
- `reviews.json` — 120 تقييم
- `stories.json` — 40 قصة نجاح
- `faq.json` — FAQ
- `share_templates.json` — قوالب المشاركة (برنامج/خطة/كويز/مجموعة)
- `quran_verses.json` — آيات (مع رقم السورة/الآية)
- `hadith.json` — أحاديث (مع المصدر)
- `assistant_content.json` — ترحيب + أزرار مساعد
- `group_templates.json` — اقتراحات أسماء + قواعد المجموعة

## بنك الأسئلة + فهرس الدورة

- `assets/data/questions.json` — بنك أسئلة (800 سؤال) مستخدم في الاختبار والكويزات.
- `assets/data/index_course.json` — فهرس الدورة (Items) يُستخدم لبناء الجدول.

> تقدر تستبدل الملفين لاحقًا بنفس البنية بدون تعديل كبير بالكود.

## القفل (محاولة واحدة كل 24 ساعة)

- يطبق على `test.html`.
- إذا انتهيت من الاختبار، أي محاولة جديدة قبل 24 ساعة تظهر لك عدّاد واضح.

## تحميل PDF

زر **تحميل PDF** في `results.html` يفتح وضع الطباعة.
- على الكمبيوتر: اختر **Save as PDF**
- على الجوال: اختر **طباعة / حفظ كملف** (حسب النظام)

داخل PDF فقط يظهر:
- شعار الأكاديمية
- اسم الطالب/ة
- رابط الدورة المكثفة

## رابط الدورة المكثفة

يتم التحكم فيه من:

`assets/js/app.js` → `APP.courseWebsite`

## الذكاء الاصطناعي (Proxy آمن)

المشروع جاهز حاليًا بدون API.
لو تبغى تفعّل الذكاء الاصطناعي لاحقًا **بدون كشف المفتاح**:

- استخدم أحد القوالب الجاهزة في `serverless_templates/`:
  - Netlify Functions
  - Vercel Serverless
  - Cloudflare Worker

**ضع المفتاح في Environment Variables باسم:**

`OPENAI_API_KEY`

> لا تضع المفتاح داخل الواجهة الأمامية.
